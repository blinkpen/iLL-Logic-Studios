Glovepie Scripts

Below is a list of my Glovepie Scripts:

Note: These scripts work for the original Wiimote and Classic Controller. The Wiimote Plus & Classic Controller Pro will not work. Third‑party versions have not been tested.

To properly use these scripts, the control settings on the emulator or VLC should be set to default. You may need WinRAR to unzip these files.

  • VLC Media Player Wiimote Media Controller : Control VLC Media Player with the Wii Remote.
  • N64 (Project 64 1.6) for Wii Classic Controller : Play N64 emulator with the classic controller.
  • N64 (Project 64 1.7) for Wii Classic Controller : Play N64 emulator with the classic controller.
  • GameCube (Dolphin) Controller for Wii Classic Controller : Play GameCube games on Dolphin with the classic controller.
  • SNES (Snes9x) for Wii Classic Controller : Play SNES emulator with the classic controller.
  • SMBX Classic Controller Script : Play SMBX with the classic controller.

These scripts work with the default controls of their respective programs. Feel free to tweak them if necessary.
