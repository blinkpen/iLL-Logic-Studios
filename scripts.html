Below is a list of my Glovepie Scripts:


Note: These scripts work for the original Wiimote and Classic Controller. The Wiimote Plus & Classic Controller Pro however, will not work. Third party versions of either tier have not been tested.
To properly use these scripts, the control settings on the emulator or shortcut settings on VLC should be set to default.
  • VLC Media Player Wiimote Media Controller: Control VLC Media Player with the Wii Remote.
  • N64 (Project 64) for Wii Classic Controller: Play N64 emulator with the classic controller.
  • GameCube (Dolphin) Controller for Wii Classic Controller: Play Gamecube games on Dolphin with the classic controller.
  • SNES (Snes9x) for Wii Classic Controller: Play SNES emulator with the classic controller.
  • SMBX Classic Controller Script: Play SMBX with the classic controller.

These scripts work with the default controls of their respective programs. Feel free to tweak them if necessary.
