How to Add a New Project
This site uses a simple three‑part system for adding new projects. Follow this structure and everything will automatically connect to the modal system.
1. Add a Project Card

Inside the correct category, add a new <li class="project"> block. This controls how the project appears in the list. 

<li class="project">
  <div class="project-header-row">
    <span class="project-name">Project Name</span>
    <button class="project-button" data-project="project-key">
      <span>✦</span> More details
    </button>
  </div>
  <p class="project-summary">Short description.</p>
</li>
2. Register the Project in JavaScript

Add the project to both maps in script.js: projectTitleMap and templateMap. 

projectTitleMap["project-key"] = "Project Name";
templateMap["project-key"] = "tpl-project-key";
3. Create the Modal Template

Each project has a matching <template> block. This is the content shown inside the modal. 

<template id="tpl-project-key">
  <data data-glyph="✦" data-subtitle="Short subtitle"></data>

  <p>Your modal content goes here.</p>
</template>

That’s it — once these three pieces are added, the modal system automatically handles opening, closing, glyphs, subtitles, and layout.
